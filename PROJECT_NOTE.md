## 프로젝트 개요

**REVOLV** — 1인 개발 하이퍼캐주얼 아케이드 게임. "REVOLVE"에서 착안한 이름, 태그라인 "One Tap · Reverse Orbit".

- **메커니즘**: 중심 코어를 도는 오브를 탭으로 방향 반전. 스파크(◆) 수집, 스파이크(✕) 회피, 콤보 시스템
- **기술 스택**: 단일 HTML 파일(바닐라 JS + Canvas + Web Audio API). 외부 에셋 파일 없음 — BGM도 절차 생성
- **지원**: 한국어/영어(기기 언어 자동 감지, 수동 전환 가능), 모바일/PC, 세로/가로 화면
- **개발 원칙**: 솔로 개발, 인앱결제 없이 향후 광고(AdMob)로만 수익화 계획

## 게임 내 주요 시스템

- **스킬**: 무적(Shield, 7초 충전→3초 방어), 자석(Magnet, 10초 충전→전체 스파크 흡수)
- **점수 마일스톤**: 5,000(블랙홀, 궤도 70% 축소 10초) → 10,000(슈퍼노바 피버, 점수×2+스파크5개 10초) → 12,000(듀얼 오빗, 바깥 궤도 해금) → 19,000(혜성, 궤도 관통 위협)
- **튜토리얼**: 첫 판 3단계 자동 안내 + 각 마일스톤 최초 등장 시 SKIP 카드(일시정지 후 수동 닫기, 반사 탭 방지 0.5초 락) + 홈 화면 `?` 도움말 페이지
- **일시정지 + 3-2-1 카운트다운**: 튜토리얼 카드를 닫거나 일시정지 해제 시 즉시 재개 대신 카운트다운 후 재개 (유저 피드백으로 추가 — "카드 닫자마자 죽는" 문제 해결)
- **가로화면 대응**: HUD가 좌상단(점수)/우측세로(스킬버튼)로 재배치, 버튼 크기는 vh 비례
- **공유 기능**: 게임오버 화면에 SHARE SCORE(Web Share API, 결과이미지 자동생성)/COPY LINK 버튼. 다음 목표 배너("NEXT EVENT: 840 pts to Black Hole")를 시작화면·게임오버 화면에 노출
- **분석**: GA4(측정ID G-K7D00GHNNV), 커스텀 이벤트(game_start/game_over/milestone/tutorial_complete/language_select)에 platform·score·play_time·type·language 파라미터. `?dev`로 개발자 기기 영구 제외 가능(netlify/GitHub만 유효, itch는 구조상 GA4 미수신)

## 홍보 운영 체계 (별도 세션 존재)

**중요**: 홍보/마케팅은 **별도의 다른 채팅 세션**("홍보팀")에서 운영 중입니다. 이 대화(PM 세션)는 기술 개발 + 전략 판단 + 홍보팀 보고 취합을 담당합니다. 두 세션은 자동 동기화되지 않으므로, 홍보팀 보고는 사용자가 수동으로 이 세션에 복붙하고, PM 결정은 사용자가 홍보팀 세션에 복붙 전달하는 방식으로 운영됩니다.

**채널 현황**: Instagram(@playrevolv, 피드 4건, bio링크=itch), itch.io devlog 3건, Reddit(r/playmygame 게시 생존중, r/LetsPlayMyGame 리포스트, r/WebGames는 카르마 20~30 도달 시 게시 예정), 디시인사이드는 보류(재검토 조건부).

**보고 원칙**: 매일이 아니라 "의미 있는 변화가 생겼을 때" 또는 주 1회 일괄. 각 작업은 STATUS.md의 T-번호로 참조.
